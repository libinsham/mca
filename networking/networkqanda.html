UNIT I – Networking Fundamentals and Protocol Models

2 Mark Questions & Answers
  1. What is DAS?
    Direct Attached Storage (DAS) is directly connected storage without a network.
  2. Define NAS.
    Network Attached Storage (NAS) provides file-level storage over a network.
  3. What is SAN?
    Storage Area Network (SAN) provides block-level storage using high-speed networks.
  4. What is RAID?
    RAID combines multiple disks to improve performance and reliability.
  5. What is software-defined storage?
    Storage managed using software instead of dedicated hardware.
  6. Write any two advantages of RAID.
    Fault tolerance, Improved performance.
13 Mark Questions & Answers
  1. Explain DAS, NAS, SAN, and CAS with advantages and applications.
    DAS: Low cost, limited scalability.
    NAS: File-based, easy sharing.
    SAN: High performance, enterprise use.
    CAS: Content-based storage, archival systems.
  2. Explain RAID levels and configurations.
    RAID 0 – Striping
    RAID 1 – Mirroring
    RAID 5 – Striping with parity
    RAID 10 – Combination of RAID 1 and RAID 0

UNIT II – Data Link and Network Layer Protocols

2 Mark Questions & Answers
  1. What are data link layer services?
    Framing, error control, flow control, access control.
  2. What is CSMA/CD?
    Medium access method used in Ethernet.
  3. Define IP routing.
    Forwarding packets based on IP addresses.
  4. What is ARP?
    Maps IP address to MAC address.
  5. What is DHCP?
    Automatically assigns IP addresses.
13 Mark Questions & Answers
  1. Explain error detection and correction techniques.
    Parity Check, Checksum, CRC, Hamming Code.
  2. Explain routing algorithms: Distance Vector and Link State.
    Distance Vector: Uses hop count (RIP).
    Link State: Uses network topology (OSPF).

    UNIT IV – Wireless and Mobile Communication Systems

    2 Mark Questions & Answers
    1. What is Mobile IP?
      Allows movement without changing IP.
    2. What is IEEE 802.11?
      Wireless LAN standard.
    3. What is ZigBee?
      Low power wireless protocol for IoT.
    4. Define MANET.
      Network without fixed infrastructure.
    13 Mark Questions & Answers
    1. Explain wireless LAN architecture and IEEE 802.11.
      Uses AP, stations, BSS, ESS and CSMA/CA.
    2. Explain mobile communication challenges.
      Handoff, Roaming, Latency, Bandwidth limitation.

    UNIT V – Modern Network Trends, IoT, Cloud & Secure Networking

    2 Mark Questions & Answers
    1. What is IoT?
      Internet of Things connects smart devices.
    2. What is MQTT?
      Lightweight IoT messaging protocol.
    3. What is VPN?
      Secure private network over internet.
    4. Expand SDN.
      Software Defined Networking.
    13 Mark Questions & Answers
    1. Explain IoT architecture and protocols.
      Includes sensing, network, processing, application layers. Uses MQTT and CoAP.
    2. Explain modern networking challenges: 5G, SDN, NFV.
      5G – High speed, low latency.
      SDN – Control and data plane separation.
      NFV – Virtualized network functions.
