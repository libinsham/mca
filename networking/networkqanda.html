UNIT I – RELATIONS, FUNCTIONS & LATTICES
Q1. Define a Binary Relation.
Let A and B be two non-empty sets. A binary relation R from A to B is a subset of the Cartesian product A × B.
Q2. Define Equivalence Relation.
A relation R is an equivalence relation if it is reflexive, symmetric, and transitive.
Q3. Prove a relation is an equivalence relation and find equivalence classes.
Reflexive ✔, Symmetric ✔, Transitive ✔ ∴ R is an equivalence relation. Equivalence classes are formed based on related elements.
Q4. Define Partial Order Relation.
A relation that is reflexive, antisymmetric, and transitive.
Q5. Explain Partial Order Relation with example.
If a relation satisfies reflexive, antisymmetric, and transitive properties, it is a partial order relation. Hasse diagram represents it without reflexive and transitive edges.
Q6. Define Lattice and its properties.
A lattice is a poset where every pair of elements has a least upper bound and a greatest lower bound. Properties: Idempotent, Commutative, Associative, Absorption.
UNIT II – PROPOSITIONAL & PREDICATE LOGIC
Q7. Define Proposition.
A proposition is a declarative statement that is either true or false.
Q8. Construct truth table for (p ∧ q) → p.
The statement is always true for all truth values. Hence, it is a tautology.
Q9. Define PCNF and PDNF.
PCNF – Conjunction of maxterms PDNF – Disjunction of minterms
Q10. Explain Predicate Logic and Quantifiers.
Predicate logic uses predicates with quantifiers. ∀ – Universal Quantifier, ∃ – Existential Quantifier.
Q11. Explain Mathematical Induction.
Steps: Base case, Induction hypothesis, Induction step. Used to prove statements for all natural numbers.
UNIT III – RECURRENCE RELATIONS
Q12. Define Recurrence Relation.
A recurrence relation defines a sequence using its previous terms.
Q13. Solve recurrence using characteristic roots.
Form characteristic equation → find roots → write general solution.
Q14. Solve an inhomogeneous recurrence relation.
Find complementary solution and particular solution to get final answer.
Q15. Explain Generating Function.
A generating function is a power series whose coefficients represent a sequence.
UNIT IV – GRAPH THEORY
Q16. Define Graph.
A graph G = (V, E) consists of vertices V and edges E.
Q17. Explain Euler Graph and Fleury’s Algorithm.
An Euler graph has all vertices of even degree. Fleury’s algorithm avoids bridges until necessary.
Q18. Explain Hamiltonian Path and Circuit.
Hamiltonian path visits each vertex once. Circuit starts and ends at the same vertex.
Q19. Explain Adjacency and Incidence Matrix.
Adjacency matrix shows vertex connections. Incidence matrix shows vertex–edge relationship.
UNIT V – GRAMMARS & AUTOMATA
Q20. Define Alphabet, String, and Language.
Alphabet – finite set of symbols String – sequence of symbols Language – set of strings
Q21. Define DFA.
A DFA is a 5-tuple (Q, Σ, δ, q₀, F).
Q22. Explain NFA and ε-NFA.
NFA allows multiple transitions. ε-NFA allows transitions without input symbols.
Q23. Prove equivalence of DFA and NFA.
For every NFA, there exists an equivalent DFA using subset construction.
Q24. Explain Minimization of Finite Automata.
Minimization reduces the number of states without changing the accepted language.
